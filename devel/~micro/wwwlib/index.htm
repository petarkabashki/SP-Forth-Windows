literal.f

Расширенный вариант ?SLITERAL-а, написанный с использованием предикатов. Размещён здесь в основном для демонстрации возможностей. Требуется: ~micro/lib/predicat.f.

predicat.f

Реализация предикатов. Каждый предикат представляет собой слово, берущее со стека несколько чисел и возвращающее флаг. Эта библиотека позволяет составлять списки предикатов. Списки фиксированной максимальной длины. При выполнении списка выполняется каждый входящий в него предикат по-очереди. Если какой-то предикат выдаёт true, слово завершается.
Для примера смотрите literal.f
i j P: name word1 word2 ... ;
Составляет список предикатов. Каждый требует j чисел на стеке, дополнительно резервируется место для добавления ещё i предикатов.
PADD ( xt v -- ) - добавление предиката к хвосту списка. v - xt списка.
PSHOW ( v -- ) - показывает список.
PREMOVE ( v -- ) - убирает последний предикат.
PREPLACE ( no-xt what-xt v -- ) - заменяет предикат № no-xt предикатом с xt=what-xt в списке v.
Требуется: ~micro/lib/stack/mem.f, ~micro/lib/parser/word.f.

replace.f

Переопределение скомпилированных слов типа : .
REPLACE: existing_word word1 word2 ... INHERITED word3 word4 word5 ... ;
Пример:
REPLACE: . DUP INHERITED INHERITED ;
1 .
1 1  Ok

REPLACE: TYPE [CHAR] { EMIT INHERITED [CHAR] } EMIT ;
TYPE isn't unique
{ Ok}
.( QWERTY)
{QWERTY}{ Ok}
